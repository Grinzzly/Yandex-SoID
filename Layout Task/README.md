# Usage

## Installation

```
npm install

# hot reload at localhost:8080
npm run dev

# build for production
npm run build

# build for production with the bundle analyzer report
npm run build --report
```

## Tests
```
# run unit tests
npm run unit

# run all tests
npm test
```
